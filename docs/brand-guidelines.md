# TinyHarvest Brand Guidelines

## Logo Variants

| File | Use Case |
|------|----------|
| `logo-icon.png` (Logo 3) | Square spaces: social avatars, favicon, profile pictures |
| `logo-horizontal.png` (Logo 2) | Wide spaces: website header, banners, email header |
| `logo-stacked.png` (Logo 1) | Alternative stacked layout if needed |

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Cream | `#F2F0E4` | Background, negative space |
| Leaf Green | `#6DB83A` | Primary brand color, sprout icon |
| Charcoal | `#2B2B2B` | Text, wordmark |

## Typography

- **Wordmark font:** Rounded, hand-lettered style (bold, uppercase, friendly)
- **Body (TBD):** Align with Modern Sprout inspo — clean sans-serif

## Brand Personality

- Fresh, approachable, natural
- Small-scale, personal, intentional growing
- Warm and friendly — not overly corporate
- Eco-conscious, sustainability-minded

## Inspiration

- [Modern Sprout](https://modernsprout.com/) — premium botanical minimalism
